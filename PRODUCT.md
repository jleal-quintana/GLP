# Capitulo IV Add-in

## Producto

Add-in moderno de Excel para descargar historico de produccion de Capitulo IV, seleccionar una o varias areas, generar hojas por area, proyectar cada area por separado y construir un resumen consolidado por suma de proyecciones individuales.

## Usuarios

Equipo tecnico y economico de Quintana Energy. El uso principal es preparar analisis de areas propias y oportunidades de adquisicion, con datos auditables dentro del workbook y graficos presentables para direccion.

## Principios

- Excel es el entregable. El add-in configura, descarga y arma; el workbook queda auditable.
- El detalle pozo-mes debe quedar disponible en hojas visibles.
- Las proyecciones se configuran desde el taskpane al crear, pero los supuestos quedan claramente editables en Excel.
- El debug debe ser visible desde el inicio en una hoja `CapIV_Debug`.
- El consolidado es la suma de areas, no un modelo independiente.
